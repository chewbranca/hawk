# hawk

A simple command line histogram tool written in AWK.


# Examples

```
python -c "from __future__ import print_function; import numpy;
[print(int(round(i))) for i in numpy.random.normal(loc=100.0, scale=10.0,
size=1000)]" | sort -n | ./hawk
```
